жҜҸдёҖдёӘExecutionJobVertexйғҪеҜ№еә”зқҖдёҖжү№ExecutionVertex(д№ҹе°ұжҳҜsubTask), еҸҜд»ҘзңӢеҲ°, иҝҷйҮҢиҪ®иҜўжҜҸдёӘExecutionVertexиҝӣиЎҢз”іиҜ·дёҖдёӘslotгҖӮ

иҜҘеҮҪж•°дё»иҰҒеҒҡдәҶдёӨд»¶дәӢжғ…:

  • еңЁcalculatePreferredLocationsдёӯзЎ®е®ҡд»ҺиҜҘsubTaskеҜ№еә”ExecutionJobVertexзҡ„жүҖжңүдёҠжёёдёӯжүҫеҲ°жңҖеҗҲйҖӮзҡ„дёҠжёёвҖқеҒҸеҗ‘дҪҚзҪ®йӣҶеҗҲвҖқгҖӮ
  • йҖҡиҝҮSlotPool$ProviderAndOwner.allocateSlotз»§з»ӯзЎ®е®ҡд»ҺвҖқеҒҸеҗ‘дҪҚзҪ®йӣҶеҗҲвҖқжүҫеҲ°дёҖдёӘе…ұдә«slotгҖӮ
    жҲ‘д»¬зҹҘйҒ“, еӨҡдёӘsubTaskе…Ғи®ёе…ұдә«slot, з»ҶиҠӮеҗҺйқўдјҡиҜҰз»ҶжҸҸиҝ°гҖӮ йӮЈд№ҲеҪ“еүҚsubTaskдёҺе“Әдәӣе·Із»ҸеҲҶй…Қзҡ„subTaskе…ұдә«slotе‘ў? дёӢжёёsubTaskдёҺе“ӘдёӘдёҠжёёsubTaskе…ұдә«slotе‘ў?

flinkдјҡж №жҚ®subTaskдёҠжёёslotзҡ„еҲҶй…ҚжқҘзЎ®е®ҡеҪ“еүҚslotзҡ„еҲҶй…Қ:

еӨ„зҗҶиҝҮзЁӢеҰӮдёӢ:

  • иӢҘиҜҘExecutionVertexжІЎжңүдёҠжёё(дҫӢеҰӮsource), йӮЈд№Ҳиҝ”еӣһдёәз©ә, жІЎжңүвҖқеҒҸеҘҪдҪҚзҪ®йӣҶеҗҲвҖқ, д№ӢеҗҺе°Ҷз”іиҜ·ж–°зҡ„slotгҖӮ
  • иӢҘеҪ“еүҚExecutionVertexжңүеұһдәҺдёҚеҗҢJobVertexеӨҡдёӘExecutionJobVertexзҡ„дёҠжёё, йӮЈд№ҲеҪ“еүҚsubеҲҶй…ҚеҲ°е“Әдәӣе…ұдә«slotзҡ„еҸҜйҖүи·Ҝеҫ„еҸӘиғҪжҳҜ: еұһдәҺеҗҢдёҖдёӘJobVertexзҡ„дёҠжёёиҠӮзӮ№дёӘж•°жңҖе°‘гҖӮдёҠеӣҫзҡ„иҜқ, е°ұдјҡйҖүжӢ©source2зҡ„жүҖжңүsubTaskдҪңдёәвҖқеҒҸеҘҪдҪҚзҪ®йӣҶеҗҲвҖқгҖӮжҲ‘д»¬жҺҘдёӢжқҘзңӢз¬¬дәҢжӯҘ, жңҖз»Ҳдјҡиҝӣе…ҘеҲ°allocateSharedSlotеҶіе®ҡsubTaskеҲҶй…ҚеҲ°е“ӘдәӣвҖқеҒҸеҘҪдҪҚзҪ®йӣҶеҗҲвҖқйҮҢslotдёҠгҖӮ

иҜҘеҮҪж•°дё»иҰҒеҒҡдәҶдёӨд»¶дәӢ:

  1. йҖҡиҝҮallocateMultiTaskSlotдә§з”ҹMultiTaskSlotLocality, йҮҢйқўеҢ…еҗ«д»ҺвҖқеҒҸеҗ‘дҪҚзҪ®йӣҶеҗҲвҖқдёӯйҖүеҸ–зҡ„йғЁзҪІеҪ“еүҚsubTaskе…ұдә«зҡ„slotгҖӮ
  2. дә§з”ҹSingleTaskSlot, еҪ“еүҚSingleTaskSlotдҪңдёәMultiTaskSlotзҡ„дёҖдёӘеӯҗеҸ¶еӯҗиҠӮзӮ№гҖӮ
    еҶҚз»§з»ӯи·ҹиҝӣд»Јз ҒеүҚ, жҲ‘д»¬йңҖиҰҒдәҶи§ЈдёӨдёӘеҸҳйҮҸresolvedRootSlotsгҖҒunresolvedRootSlotsгҖӮе…ұдә«slotйғҪдјҡд»ҺиҝҷдёӨдёӘеҸҳйҮҸдёӯиҺ·еҸ–, иҝҷдёӨдёӘеҸҳйҮҸдёәе…ұдә«з»„жүҖжӢҘжңү, й»ҳи®Өе…ұдә«з»„дёәdefaultгҖӮ
    unresolvedRootSlots: еҪ“еҪ“еүҚsubTaskжӯЈеңЁзЎ®и®ӨйғЁзҪІеҲ°йӮЈдёӘslotдёӯж—¶, дјҡе°ҶиҜҘslotдҝқеӯҳеңЁunresolvedRootSlots; еҪ“зЎ®е®ҡеҘҪйғЁзҪІеҲ°е“ӘдёӘslotж—¶, дјҡе°ҶиҜҘдҝЎжҒҜд»ҺunresolvedRootSlotsдёӯз§»йҷӨ, е№¶ж”ҫе…ҘresolvedRootSlotsдёӯ еҪ“жҲ‘д»¬жҹҘжүҫжҳҜеҗҰжңүеҸҜеҲ©з”Ёзҡ„slotж—¶, дјҡд»ҺиҝҷдәӣеҸҳйҮҸдёӯжҹҘжүҫгҖӮ
    жҲ‘д»¬еҶҚиҝӣе…ҘжӯЈйўҳ, зңӢallocateMultiTaskSlotзңӢжҳҜеҰӮдҪ•з»ҷsubTaskеҲҶй…Қslotзҡ„:

иҜҘеҮҪж•°дё»иҰҒйҖ»иҫ‘еҰӮдёӢ:

  • д»ҺresolvedRootSlotsгҖҒunresolvedRootSlotsдёӯжҹҘжүҫжҳҜеҗҰжңүеҸҜе…ұдә«зҡ„slotгҖӮ

  • е°Ҷз”іиҜ·зҡ„slotдҝЎжҒҜд№ҹеӯҳж”ҫе…ҘunresolvedRootSlotsдёӯ, зӯүжҲҗеҠҹз”іиҜ·еҗҺеҶҚеӯҳж”ҫе…ҘresolvedRootSlotsгҖӮ
    жҲ‘д»¬еҶҚжҺҘзқҖзңӢжҳҜеҰӮдҪ•еҗ‘ResourceManagerз”іиҜ·TaskManagerзҡ„гҖӮ

еҸҜд»ҘзңӢеҲ°:

  • иӢҘе·Із»ҸеҲқе§ӢеҢ–д№ӢеҗҺ, дјҡеҺ»еҗ‘ResourceManagerз”іиҜ·TaskManagerгҖӮ

  • йҖҡиҝҮfindMatchingSlotжЈҖжҹҘжҳҜеҗҰжңүзҺ°жҲҗеҸҜз”Ёзҡ„slot, е…¶дёӯfreeSlotsеҢ…еҗ«зқҖavailiable slot. жҜ”еҰӮеңЁжҜҸеҪ“жңүж–°зҡ„TaskManagerеҗ‘JobManagerжіЁеҶҢж—¶, е°ұдјҡи°ғз”ЁSlotManager.registerSlotRequest(), еңЁfreeSlotsдёӯжіЁеҶҢиҜҘTMеҸҜз”Ёзҡ„slotгҖӮиӢҘжңүеҸҜз”Ёslotж—¶еҖҷ, е°ұдјҡи°ғз”ЁallocateSlotиҝӣиЎҢеҲҶй…ҚгҖӮ
  • иӢҘжІЎжңүеҸҜз”Ёз©әй—Іslot, йҖҡиҝҮallocateResourceз”іиҜ·TM, жңҖз»Ҳдјҡи°ғз”ЁYarnResourceManager.requestYarnContainerиҝӣиЎҢз”іиҜ·гҖӮ

LinkedTransferQueueеҺҹзҗҶеҲҶжһҗ

2018-12-15

LinkedTransferQueueдҪңдёәж— иҫ№з•Ңзҡ„йҳ»еЎһйҳҹеҲ—, еҗҢж—¶з»§жүҝдәҶTransferQueueе’ҢAbstractQueueгҖӮ зӣёиҫғдәҺе…¶д»–зҡ„йҳ»еЎһйҳҹеҲ—: LinkedTransferQueueзү№ж®Ҡд№ӢеӨ„еңЁдәҺTransferQueueжҺҘеҸЈгҖӮ TransferQueueиҜҙжҳҺеҰӮдёӢ:

  • Transfers the element to a waiting consumer immediately, if possible. More precisely, transfers the specified element immediately if there exists a consumer already waiting to receive it (in {@link #take} or timed {@link #poll(long,TimeUnit) poll}), otherwise returning {@code false} without enqueuing the element.

иҜҙзҡ„жҳҜLinkedTransferQueueдёҚд»…е®һзҺ°дәҶжҷ®йҖҡBlockingQueueзҡ„еҠҹиғҪ, еҸҰдёҖдёӘдјҳзӮ№е°ұжҳҜ: еҪ“жңүж¶Ҳиҙ№иҖ…зӯүеҫ…ж•°жҚ®ж—¶, з”ҹдә§иҖ…еҸҜд»ҘзӣҙжҺҘе°Ҷж•°жҚ®дәӨз»ҷж¶Ҳиҙ№иҖ…иҖҢдёҚжҳҜеҶҚиҝӣе…ҘйҳҹеҲ—гҖӮ дёҺLinkedBlockingQueueзӣёжҜ”, LinkedBlockingQueueеңЁtakeе’Ңputж“ҚдҪңж—¶, йғҪжҳҜйҖҡиҝҮlockжқҘжҺ§еҲ¶, еҪ“й«ҳе№¶еҸ‘ж“ҚдҪңtakeе’Ңputж“ҚдҪң, й”Ғзҡ„иҺ·еҸ–е’ҢйҮҠж”ҫйғҪжҳҜжҜ”иҫғеҪұе“ҚжҖ§иғҪзҡ„гҖӮ иҖҢLinkedTransferQueueеҜ№иҝҷз§ҚдҪҝз”ЁиҝӣиЎҢдәҶж”№иҝӣ, еҪ“з”ҹдә§иҖ…еӯҳж”ҫж•°жҚ®ж—¶, еҸ‘зҺ°жңүж¶Ҳиҙ№иҖ…зӯүеҫ…ж¶Ҳиҙ№ж•°жҚ®, з”ҹдә§иҖ…еҸҜд»Ҙи°ғз”ЁtransferзӣҙжҺҘе°Ҷж•°жҚ®дәӨз»ҷж¶Ҳиҙ№иҖ…, иҖҢдёҚз”ЁйҖҡиҝҮйҳ»еЎһйҳҹеҲ—жқҘдј йҖ’ж•°жҚ®, еҮҸе°‘дәҶй”Ғзҡ„йҮҠж”ҫдёҺиҺ·еҸ–гҖӮ

LinkedTransferQueueзұ»з®Җд»Ӣ

LinkedTransferQueueзҡ„йҳ»еЎһйҳҹеҲ—зҡ„иҠӮзӮ№Nodeи®ҫи®ЎеҰӮдёӢ:

LinkedTransferQueueдёӯзҡ„йҳ»еЎһйҳҹеҲ—иҠӮзӮ№еҲҶдёәдёӨз§Қ, ж•°жҚ®иҠӮзӮ№е’ҢиҜ·жұӮиҠӮзӮ№гҖӮеҪ“з”ҹдә§иҖ…и°ғз”Ёputж—¶, иӢҘжІЎжңүж¶Ҳиҙ№иҖ…йңҖиҰҒж•°жҚ®, зӣҙжҺҘе°Ҷж•°жҚ®еӯҳж”ҫе…Ҙйҳ»еЎһйҳҹеҲ—дёӯ, йӮЈд№Ҳе°Ҷз”ҹжҲҗж•°жҚ®иҠӮзӮ№(isData=trueгҖҒitemдёәз”ЁжҲ·еӯҳж”ҫзҡ„ж•°жҚ®, waiterдёәиҜҘиў«йҳ»еЎһзҡ„зәҝзЁӢ)гҖӮеҪ“и°ғз”ЁtakeиҠӮзӮ№ж—¶, иӢҘжІЎжңүж•°жҚ®еҸҜдҫӣеҸ–иө°, йӮЈд№Ҳе°ҶеңЁйҳ»еЎһйҳҹеҲ—дёӯе»әз«ӢиҜ·жұӮиҠӮзӮ№(isData=falseгҖҒitemдёәnull, waiterдёәиҜҘиў«йҳ»еЎһзҡ„зәҝзЁӢ)гҖӮиӢҘйҳ»еЎһйҳҹеҲ—жңӘеҢ№й…ҚиҠӮзӮ№дёәиҜ·жұӮиҠӮзӮ№, ж–°жқҘзҡ„д№ҹдёәиҜ·жұӮ, йӮЈд№Ҳе°ҶйҖҡиҝҮе°ҫеҠ зҡ„ж–№ејҸеӯҳж”ҫ; иӢҘж–°жқҘдёҖдёӘж•°жҚ®иҠӮзӮ№, йӮЈд№Ҳе°ұеҸҜд»Ҙе’Ңйҳ»еЎһйҳҹеҲ—дёӯзҡ„иҠӮзӮ№еҢ№й…Қ, ж•°жҚ®е°ұдёҚдјҡж”ҫеҲ°йҳ»еЎһйҳҹеҲ—дёӯгҖӮжіЁж„Ҹ: йҳ»еЎһйҳҹеҲ—дёӯжңӘеҢ№й…ҚиҠӮзӮ№жЁЎејҸдёҖиҮҙ, иҖҢдёҚжҳҜиҜҙйҳ»еЎһйҳҹеҲ—жүҖжңүиҠӮзӮ№жЁЎејҸдёҖиҮҙгҖӮ йҳ»еЎһйҳҹеҲ—дёӯиў«еҢ№й…Қзҡ„иҠӮзӮ№NodeеҸҳеҢ–жғ…еҶөеҰӮдёӢ:
